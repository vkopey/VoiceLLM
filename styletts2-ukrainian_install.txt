Завантажити і розпакувати https://www.python.org/ftp/python/3.10.11/python-3.10.11-embed-amd64.zip
Завантажити get-pip.py: https://bootstrap.pypa.io/get-pip.py
Запустити cmd.exe та запустити з розпаковоної папки:
cd /D f:\python-3.10.11-embed-amd64_TTS\
python get-pip.py
Додати у файл python310._pth рядки:
Lib
Lib\site-packages
або розкоментувати рядок #import site

Установити пакети:
cd scripts
pip3 install -r requirements.txt
git clone https://huggingface.co/spaces/patriotyk/styletts2-ukrainian

Заставити Python використовувати UTF-8 постійно
set PYTHONUTF8=1
або
python -X utf8 app.py

Запуск локально з веб інтерфейсом
python app2.py
Це перший раз установлює додаткові пакети і завантажує моделі (700МБ) в папку на диску С

Запуск локально
python app3.py
